ANAGRAMA

Um anagrama é uma palavra ou frase formada pela reorganização das letras de uma palavra ou frase diferente, normalmente usando todas as letras originais exatamente uma vez.

Exemplo 1: "abc" é um anagrama de "cba".

Exemplo 2: "uva" é um anagrama de "uav", "vua", "vau", "auv", "avu".
